# FileScanApp

C# .netcore Console app
Read an input file BDG_Input.txt(located in the TextInputFile Directory in the root of the project).
Convert the file data to JSON format.
Write a new output file BDG_Output.json with JSON formatted data(BDG_Output.json will be written to the C drive of the machine the application is running on.).
logs are wriiten in the logs directory in the root.
